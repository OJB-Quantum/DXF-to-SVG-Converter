# DXF-to-SVG-Converter
A tool that runs in the browser using Colab and a GPU to identify and convert a DXF layout into an SVG output. Control knobs are included. The Colab code is open source, written by Onri Jay Benally in 2026.

---

You can use a free GPU in Google Colab as needed.
